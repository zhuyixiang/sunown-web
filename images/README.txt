============================================
IMAGE TIPS
============================================
1. Use high-quality images (at least 72dpi for web)
2. Optimize images for web to reduce file size
3. Use JPG for photos, PNG for logos with transparency
4. Keep consistent aspect ratios within each category
5. Use warm, professional imagery that matches the brand colors
